High-performance post favorites with REST API and Redis-friendly object caching.

== Description ==

Favorite Posts Performance Plugin adds a Save/Saved button for posts. It supports authenticated users, nonce-protected REST requests, optimistic frontend UX, and object-cache-backed favorite state retrieval.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate via the Plugins menu.
3. Add `.fp-favorite-btn` buttons in your templates.
